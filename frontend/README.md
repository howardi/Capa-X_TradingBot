# CapaRox Trading Bot Frontend

This directory contains the React-based frontend for the CapaRox Trading Bot.

## Setup

1.  Navigate to this directory:
    ```bash
    cd frontend
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

## Development

To start the development server:

```bash
npm run dev
```

## Build

To build for production:

```bash
npm run build
```
